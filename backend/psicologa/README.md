# Sistema de Clínica - API

Este projeto é uma API REST desenvolvida em Java com Spring Boot para gerenciamento de atendimentos clínicos.

## Funcionalidades

* Cadastro de pacientes
* Agendamento e reagendamento de consultas
* Visualização de agenda diária e semanal
* Controle de eventos
* Envio de lembretes via WhatsApp (Twilio)

## Tecnologias utilizadas

* Java
* Spring Boot
* JPA / Hibernate
* MySQL
* HTML, CSS e JavaScript
* Postman

## Como rodar o projeto

1. Clone o repositório:
   git clone https://github.com/seu-usuario/sistema-clinica-api.git

2. Configure o arquivo `application.properties` com base no `application-example.properties`

3. Configure as variáveis de ambiente:

* DB_URL
* DB_USER
* DB_PASSWORD
* TWILIO_SID
* TWILIO_TOKEN

4. Execute a aplicação

## Observação

Este projeto utiliza dados fictícios para fins de estudo e demonstração.
